"H1" cria titulos

"H2" cria subtitulos, e assim por diante até h6

"P" cria paragrafos

"Img" adiciona uma imagem

"Audio" adiciona audio na página

OBS:O autoplay não esta funcionando no chrome

"Video" adiciona um video na página e "width" define o tamanho da ocupação dele na tela

"Poster" permite adicionar uma imagem como capa do video

OBS:O autoplay não esta funcionando no chrome

"Picture" permirte usar uma imagem como uma picture onde posso usar ela como um link

É possivel adicionar estilo na página por meio da tag "Style", porem é melhor utilizar o CSS, por ser mais prático do que adicionar "style" em cada tag do HTML
