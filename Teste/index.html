OBS: ISSO NÃO É TAMANHO DE FONTE!

Para páragrafos utilizamos a tag "p".

Para links utilizamos a tag "link".

Para anexar uma imagem utilizamos a tag "img".

OBS: Podemos também, anexar uma imagem direto da internet, basta colar o link da imagem.

Mudanças de estilo da pagina (Cor, Posição da imagem, fonte, tamanho da fonte,etc.) não são feitas mais em "HTML" e sim no CSS!

Algumas tags deixaram de funcionar ou Foram substituidas, como "big" e "u".

Para fazer uma quebra de texto assim,
uitilizamos a tag "br".

Para usar o simbolo sem transformar em tag usamos assim (olhar no código), <p> e <p>.

Para deixar um texto em Negrito, usamos a tag "strong".

Se quisermos marcar um texto, usamos a tag "mark", porém se quisermos mudar a cor da marcação, temos que colocar uma "style"
entre "Title" e "Head".

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Para deixar um texto sobrescrito assim, x20+3 usamos a tag "sup".

Para deixar um texto subscrito assim, H20 usamos a tag "sub".
